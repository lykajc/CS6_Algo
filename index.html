Proposed System

IBFO-ODLAD Pipeline

The IBFO-ODLAD system integrates four sequential stages for automated anomaly detection in IoT networks (Khayyat, 2023).

1

Z-Score Normalization

Standardizes input features to mean 0, SD 1

2

IBFO Feature Selection

Selects optimal feature subset via SA-enhanced BFO

3

MLSTM Classification

Multiplicative LSTM classifies normal vs. anomalous traffic

4

BOA Hyperparameter Tuning

Bayesian optimization selects optimal MLSTM hyperparameters

Algorithm Comparison

Baseline vs. Enhanced Algorithm

The IBFO improves upon the standard BFO by integrating Simulated Annealing, enabling probabilistic escape from local optima during feature subset selection (Khayyat, 2023).

Baseline
🦠

Standard BFO

Bacterial Foraging Optimization

  • Chemotaxis: swim and tumble via flagella motion
  • Swarming: cell-to-cell attraction and repulsion
  • Reproduction: poorest half eliminated, best half cloned
  • Elimination-dispersal: random reinitializations with probability Ped
  • Greedy acceptance — only improves, never explores worse states
  • Prone to local optima trapping on complex fitness landscapes

Complexity

TimeO(Ned·Nre·Nc·M·D)
SpaceO(M·D)
ConvergenceSuboptimal (local minima)
Enhanced
⚡

Improved BFO (IBFO)

BFO + Simulated Annealing

  • All BFO operators retained (chemotaxis, swarming, reproduction)
  • SA Metropolis condition added within chemotaxis step
  • Worse solutions accepted with probability exp(−ΔE/T)
  • Temperature T decreases geometrically: T ← T·α at each step
  • Probabilistic escape from local optima — key enhancement
  • Better convergence to global optimum on high-dimensional data

Complexity

TimeO(Ned·Nre·Nc·M·D)
SpaceO(M·D) + O(1) for SA
ConvergenceNear-global optimum

Comparative Analysis

BFO vs. IBFO — Key Findings

The IBFO consistently outperforms the standard BFO across all evaluated metrics, primarily due to the Simulated Annealing enhancement.

Key Takeaway

The Simulated Annealing enhancement in IBFO reduces classification error and selects fewer, more informative features than standard BFO. On the UNSW NB-15 dataset, IBFO-ODLAD achieves 98.89% accuracy — outperforming all compared baselines including SSA-CRNN (98.25%) and VLSTM (95.73%) — while selecting only 19 of 42 available features (Khayyat, 2023).
